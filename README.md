## Compile commands for all programs in Jupyter Notebook

- For C Programs only  
    ```
    gcc filename.c  
    ./a.out filename
    ```

- For C++ Programs only  
    ```
    g++ filename.cpp  
    ./a.out filename
    ```

- For Java Programs only  
    ```
    javac filename.java  
    java filename
    ```

- For LEX Programs only  
    ```
    flex filename.l  
    gcc lex.yy.c  
    ./a.out
    ```


- For BISON Programs only
    ```
    yacc filename.y  
    gcc y.tab.h  
    ./a.out
    ```
